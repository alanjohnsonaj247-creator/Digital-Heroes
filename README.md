# KTU CGPA & Percentage Calculator

A simple, fast, and user-friendly web application that helps KTU students calculate their overall CGPA and percentage based on semester SGPA values.

## About the Project

As a KTU engineering student, I often needed a quick way to calculate my cumulative CGPA and equivalent percentage without manually performing calculations every semester. This tool was created to simplify that process and provide instant, accurate results.

The application allows users to enter SGPA values for up to eight semesters and automatically calculates:

- Overall CGPA
- Equivalent Percentage
- Instant Results

## Features

- Clean and responsive user interface
- Supports Semester 1 to Semester 8 SGPA inputs
- Automatic CGPA calculation
- Percentage conversion based on KTU formula
- Works on desktop and mobile devices
- Fast and lightweight
- No installation required

## Calculation Formula

### CGPA

Overall CGPA is calculated as the average of the entered semester SGPA values.

### Percentage

Percentage = (CGPA - 0.75) × 10
